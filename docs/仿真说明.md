# mex 编译说明

mex 环境配置：
参考文章：[使用 CMake 编译 matlab mex 文件](https://pumpkin_rice.gitlab.io/p/4688c588c2ee3ac78aec4dca2ed2ccdd17c6421a986ed14b8fbd27e9fddcd8dc/)

在顶层目录下运行 `cmake --build build` 即可，编译完成后，mex 文件会出现在顶层目录下。

## pica-drive 与 模型比较
可以运行模型 mcb_pmsm_foc_abs_current_loop_mex_compare.slx 查看输入与输出进行比较。

当前行为基本一致。

**注意：**
- pica-drive 与 模型 实际仍有不同，pica-drive 中对于电流和积分饱和的限制更完善，模型中则无此部分；
- pica-drive 来源于 ODrive，详细可以对照模型、pica-drive 查看 ODrive 代码（其代码较为分散）
- 每次使用 simulink 进行仿真后，需要运行 `clear mex_picadrive_bldc_foc` 清除缓存，否则下次仿真时会数据错误，此问题与 matlab 有关
